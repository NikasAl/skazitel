# Roadmap Сказителя

## Стратегия

1. **Сначала Chrome-плагин** — быстрее разрабатывается, проще итерировать, мгновенно доступен для тестирования на себе
2. **Отладка функционала** — проходим все типы упражнений, настраиваем промпты, проверяем UX
3. **Порт на Android** — ориентируемся на проверенный плагин, переносим концепт в Kotlin + Compose
4. **Публикация** — RuStore (Android), Chrome Web Store (плагин)
