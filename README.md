# Сказитель

> Тренажёр стихосложения с ИИ-наставником. Учись писать стихи через геймификацию: ритм, рифма, метафора, фонетика.

## О проекте

**Сказитель** — приложение для обучения ремеслу написания стихов с помощью нейросетей. Не генератор, который пишет за тебя, а тренажёр, который развивает конкретные навыки: метрическое чутьё, рифмический словарь, конструкция метафор, управление формой.

Все данные хранятся локально. Нейросети подключаются через API-ключ пользователя (OpenRouter, GigaChat, z-ai) — приложение не имеет собственного бэкенда.

### Почему «Сказитель»

Сказитель — устный поэт, исполнитель былин и эпических сказаний. Архетип живого слова, который встаёт на путь мастерства. Это название отражает суть: не инструмент для настройки, а путь к умению формулировать мысли своей головой.

## Возможности

- **12 типов упражнений** — 8 творческих (ритм, рифма, метафора, ограничения, деконструкция, фонетика, проза→поэзия, анти-клише) и 4 дрили (слоги, ударения, рифмы, конструктор строки)
- **Дрели** — быстрые задания с выбором из вариантов, работают без API-ключа
- **Анализ текста в реальном времени** — слоги, ударения, определение стихотворного размера (9 метров) при вводе
- **Управление темами** — создание, выбор активной темы, подсчёт упражнений по теме
- **Библиотека упражнений** — фильтрация, повтор, редактирование, удаление
- **«Мои стихи»** — автосохранение творческих ответов, просмотр, удаление
- **Система сложности 1–10** — масштабирует параметры генерации
- **Геймификация** — стрики, XP, 10 уровней (от Подмастерья до Сказителя)
- **Мульти-провайдер LLM** — 21 модель через OpenRouter (15), z-ai (4), GigaChat (2)
- **Локальное хранение** — IndexedDB (Dexie.js), импорт/экспорт в JSON
- **Офлайн-режим** — 8 встроенных упражнений работают без API
- **Устойчивость к LLM** — авто-распаковка JSON, нормализация форматов, фоллбэки

## Быстрый старт (Chrome-плагин)

### Требования

- Node.js >= 18
- npm >= 9
- Chrome / Edge / Chromium-based браузер

### Установка

```bash
# Клонирование репозитория
git clone https://github.com/NikasAl/skazitel.git
cd skazitel/chrome-extension

# Установка зависимостей
npm install

# Сборка для разработки
npm run dev

# Сборка для продакшена
npm run build
```

### Загрузка в Chrome

1. Соберите проект: `cd chrome-extension && npm run build`
2. Откройте `chrome://extensions/`
3. Включите «Режим разработчика»
4. Нажмите «Загрузить распакованное расширение»
5. Выберите папку `chrome-extension/dist`
6. Откройте новую вкладку — появится Сказитель

### Настройка API

1. Пройдите онбординг (выберите уровень, введите тему)
2. Перейдите в Настройки (⚙️)
3. Выберите провайдера и введите API-ключ
4. Нажмите «Проверить ключ»
5. Выберите модель из выпадающего списка
6. Сохраните настройки и начните тренировку
