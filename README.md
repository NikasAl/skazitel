# Сказитель

> Тренажёр стихосложения с ИИ-наставником. Учись писать стихи через геймификацию: ритм, рифма, метафора, фонетика.

## О проекте

**Сказитель** — приложение для обучения ремеслу написания стихов с помощью нейросетей. Не генератор, который пишет за тебя, а тренажёр, который развивает конкретные навыки: метрическое чутьё, рифмический словарь, конструкция метафор, управление формой.

Все данные хранятся локально. Нейросети подключаются через API-ключ пользователя (OpenRouter, GigaChat, z-ai) — приложение не имеет собственного бэкенда.

### Почему «Сказитель»

Сказитель — устный поэт, исполнитель былин и эпических сказаний. Архетип живого слова, который встаёт на путь мастерства. Это название отражает суть: не инструмент для настройки, а путь к умению формулировать мысли своей головой.

## Возможности

- **12 типов упражнений** — 8 творческих и 4 дрила (базовые навыки: слоги, ударения, рифмы, конструктор строки)
- **Дрели для начинающих** — быстрые задания с выбором из вариантов, работают без API-ключа
- **Система сложности 1–10** — адаптируется под уровень пользователя
- **Геймификация** — стрики, XP, уровни, достижения
- **Мульти-провайдер LLM** — OpenRouter (14 моделей), GigaChat, z-ai через единый интерфейс
- **Локальное хранение** — все данные на устройстве через IndexedDB (Dexie.js), импорт/экспорт в JSON
- **Chrome Popup** — работает как popup-расширение (не новая вкладка)

## Быстрый старт (Chrome-плагин)

### Требования

- Node.js >= 18
- npm >= 9
- Chrome / Edge / Chromium-based браузер

### Установка

```bash
# Клонирование репозитория
git clone https://github.com/NikasAl/skazitel.git
cd skazitel/chrome-extension

# Установка зависимостей
npm install

# Сборка для разработки
npm run dev

# Сборка для продакшена
npm run build
```

### Загрузка в Chrome

1. Соберите проект: `cd chrome-extension && npm run build`
2. Откройте `chrome://extensions/`
3. Включите «Режим разработчика»
4. Нажмите «Загрузить распакованное расширение»
5. Выберите папку `chrome-extension/dist`
6. Нажмите на иконку расширения в тулбаре Chrome — откроется popup Сказителя

### Настройка API

1. Откройте Сказитель (кликните на иконку расширения)
2. Перейдите в Настройки
3. Выберите провайдера и введите API-ключ
4. Выберите модель из выпадающего списка
5. Начните тренировку

## Технологический стек

### Chrome-плагин

| Слой | Технология |
|------|-----------|
| UI | React 18 + TypeScript |
| Сборка | Vite |
| Стилизация | Tailwind CSS + CSS Modules |
| Хранение | IndexedDB (через Dexie.js) |

|------|-----------|
| UI | Jetpack Compose |

| Сборка | Gradle (Kotlin DSL) |
| Хранение | Room + DataStore |
| Сеть | Retrofit + OkHttp |
| Навигация | Compose Navigation |
| DI | Koin |
| Тестирование | JUnit + Compose Testing |

## Концепция работы

```
Пользователь выбирает тему
         |
    ┌────┴────┐
    │ API-ключ │         Нет API-ключа
    │ доступен? ├──────────────────────┐
    └────┬────┘                       |
         Да                           |
    LLM генерирует          Встроенные темы
    упражнение              из репозитория
         |                       |
         └───────────┬───────────┘
                     |
            Пользователь
            выполняет задание
                     |
            LLM проверяет ответ
            (или встроенная проверка)
                     |
          Фидбек + XP + прогресс
                     |
          Каждое 5-е упражнение:
          Творческая сессия
```
